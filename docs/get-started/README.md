# Get Started with Brebaje

Welcome to Brebaje! This section provides an overview of the platform and guides you through getting started.

## What is Brebaje?

Brebaje is a zero-knowledge proof ceremony management platform designed to coordinate trusted setup ceremonies for zkSNARK circuits. The platform provides:

- **Ceremony Management**: Create and manage trusted setup ceremonies
- **Participant Coordination**: Enroll and track participants
- **Contribution Tracking**: Monitor cryptographic contributions
- **Circuit Management**: Define and manage zkSNARK circuits
- **Secure Storage**: S3-based storage for ceremony artifacts

## Quick Start

1. **Installation**: Follow the [Setup Guide](../SETUP.md) to install and configure Brebaje
2. **Authentication**: Set up GitHub OAuth authentication
3. **Create a Project**: Start by creating your first project
4. **Create a Ceremony**: Set up a trusted setup ceremony
5. **Add Participants**: Enroll participants in your ceremony
